# P1

Startup/landing project built with React, TypeScript, and shadcn/ui.

**Repository**: [https://github.com/JohnFok119/P1](https://github.com/JohnFok119/P1)

## Tech stack

- **Vite** – build tool and dev server
- **TypeScript** – type-safe JavaScript
- **React 18** – UI library
- **React Router** – client-side routing
- **shadcn/ui** (Radix UI) – accessible components
- **Tailwind CSS** – styling
- **Vitest** – unit tests

## Getting started

**Prerequisites**: Node.js and npm (e.g. [install Node with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Clone the repository
git clone https://github.com/JohnFok119/P1.git
cd P1

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

## Scripts

| Command           | Description                |
|-------------------|----------------------------|
| `npm run dev`     | Start dev server (Vite)    |
| `npm run build`   | Production build           |
| `npm run preview` | Preview production build   |
| `npm run lint`    | Run ESLint                 |
| `npm run test`    | Run tests (Vitest)         |
| `npm run test:watch` | Run tests in watch mode |

## Project structure

- `src/components/` – React components (Navbar, HeroSection, AboutSection, TeamSection, etc.)
- `src/pages/` – Page-level components
- `src/components/ui/` – shadcn/ui components
- `src/assets/` – Images and static assets
- `public/` – Public static files
